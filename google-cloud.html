Google Cloud 基礎まとめ

Google CloudはGoogleが提供するクラウドサービスです。
データ分析やAI分野に強いことが特徴です。 


Google Cloud

│

├─ Compute Engine（仮想マシン）

│      ├─ ライブマイグレーション

│      └─ プリエンプティブVM

│

└─ AIサービス

       ├─ Vision API

       ├─ AutoML

       └─ AI Platform
ライブマイグレーション（Live Migration）（ライブ マイグレーション）
一言でいうと 仮想マシンを停止せずに別の物理サーバーへ移動する技術。

Google Cloudでは、 物理サーバーのメンテナンスが必要になっても、 仮想マシンを止めずに別のサーバーへ移動できます。

利用者はWebサービスを使い続けられます。 裏側ではサーバーだけ引っ越しています。
マイグレーション(Migration)とは？ 「移行・引っ越し」という意味です。 ライブマイグレーションは 「動かしたまま引っ越し」 と覚えると分かりやすいです。
プリエンプティブVM（Preemptible VM）（プリエンプティブ ブイエム）
一言でいうと 途中で停止される可能性がある代わりに、 非常に安く借りられる仮想マシン。
向いている仕事
  • 動画変換
  • 画像変換
  • AI学習
  • 大量データ処理
向いていない仕事
  • Webサーバー
  • 銀行システム
  • 24時間動かすシステム
Googleの都合で停止される可能性があるためです。
Vision API（Cloud Vision API）（クラウド ビジョン エーピーアイ）
一言でいうと 画像をAIが解析してくれるサービス。
できること
  • 文字認識（OCR）
  • 顔検出
  • 物体認識
  • ロゴ認識
  • バーコード認識
例えば レシート画像 ↓ Vision API ↓ 金額や店名を読み取る ということができます。
AutoML（オートエムエル）
一言でいうと プログラミングをほとんどせずにAIモデルを作れるサービス。
猫1000枚 犬1000枚 ↓ 学習 ↓ 猫・犬を判定するAI完成
AIの知識が少なくても利用しやすいサービスです。
AI Platform（AI Platform）（エーアイ プラットフォーム）
一言でいうと 自分でAIモデルを開発・学習・公開するためのサービス。
Pythonで作ったAI ↓ Google Cloudへ配置 ↓ 公開
AutoMLより自由度が高い反面、 機械学習の知識が必要です。 ※現在は Vertex AI（バーテックス エーアイ）へ機能が統合されています。研修資料が古い場合は「AI Platform」という名称で説明されていることがあります。

AIサービスの違い

サービス 何をする？ 対象者
Vision API 画像を解析する 誰でも
AutoML 簡単にAIを作る 初心者
AI Platform 本格的にAI開発 AIエンジニア

料金イメージ

  • Vision API：画像を何枚解析したかで課金
  • AutoML：学習時間と予測回数で課金
  • AI Platform（Vertex AI）：学習用マシン・GPU・推論時間などで課金

どれも従量課金ですが、 AI Platform（Vertex AI）が最も自由度が高い分、 料金体系も複雑です。
