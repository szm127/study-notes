キャリアプラン整理メモ（2026年7月）

1. 相談のきっかけ

  • 今後どんな仕事を目指せばよいか分からない。
  • リモート中心・9〜17時・子育てと両立できる働き方を実現したい。
  • 会社は好きだが、将来も通用するスキルや専門性が欲しい。
  • 資料作成は楽しいが、それだけを続ける将来像には少し不安がある。
  • 市場価値を示せる資格や肩書きが欲しい。

2. 自分について分かったこと

仕事で大切にしたいこと

  • 生活を整える時間を残したい。
  • 家事や将来の子育てを大切にしたい。
  • 出世よりも長く安定して働きたい。
  • 正社員で産休・育休を取りたい。

向いていること

  • 情報を整理する。
  • 教材・マニュアルを作る。
  • 仕事を分かりやすくする。
  • 標準化・仕組み化を考える。
  • ITを使って人が仕事をしやすくする。

あまり向いていないこと

  • 終わりのない技術競争。
  • 暗記中心・引っかけ問題中心の資格勉強。
  • 毎日出社・長時間労働。

3. 一番しっくりきたキャリアの方向性

「ITを使って、仕事をやりやすくする人になる。」

教材作成だけではなく、

  • 業務改善
  • ナレッジ整備
  • マニュアル作成
  • プロジェクト運営の標準化
  • AIやITツールの活用

などを通して、「会社の仕事が回りやすくなる仕組み」を作る人を目指す。

4. 資格の考え方

資格は単なる肩書きではなく、

「市場価値」と「自分自身の安心感」の両方を満たすもの

として取得する。

現時点の優先順位

  1. 情報セキュリティマネジメント試験（最優先）
  2. CAPM
  3. テクニカルライティング検定
  4. 必要に応じて応用情報・ITサービスマネージャなど

Java Silverは否定しないが、現時点では優先度を下げる。

5. 今後会社で積みたい実績

  • 教材・マニュアルの改善
  • ナレッジ整備
  • 業務の標準化
  • プロジェクト運営の改善
  • AI・ITツールを活用した効率化

「この人がいると仕事がやりやすくなる」と思われる実績を積み重ねる。

6. 上司との面談で伝えたいこと

  • 資料作成は今後も伸ばしたい。
  • その先として、仕事をやりやすくする仕組みづくりにも挑戦したい。
  • 教材・マニュアルだけでなく、業務改善やナレッジ整備にも関わりたい。
  • 会社として期待される役割や取得すると良い資格を相談したい。

7. 今後の方針

  1. まずは情報セキュリティマネジメント試験を目標にする。
  2. 勉強時間より「勉強習慣」を作ることを優先する（毎日15〜20分程度）。
  3. 資料作成を軸にしながら、改善・標準化・ナレッジ整備へ仕事の幅を広げる。
  4. 成果物が残る仕事を意識し、リモートでも価値を出せる人材になる。
  5. 5年後には「ITを使って、仕事をやりやすくする人」と説明できるキャリアを目指す。
